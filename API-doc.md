# TheMovieDB:
* Link: https://developer.themoviedb.org/docs/getting-started
### Restricciones de uso:
* API de uso gratuito para fines no comerciales.
* Se necesita una licencia para utilizar la API y/o datos/imágenes con fines comerciales.
* Se deberá utilizar el logotipo de TMDB para identificar el uso de las API.
* Se deberá colocar el aviso “Este producto utiliza la API de TMDB, pero no está respaldado ni certificado por TMDB”.
* Al atribuir la API, deberá estar dentro de la sección “Acerca de” o “Créditos”.
### Implementación:
* https://api.themoviedb.org/3/movie/: Obtiene las listas de películas.
* https://api.themoviedb.org/3/movie/now_playing: Películas en cartelera.
* https://api.themoviedb.org/3/movie/popular: Películas populares.
* https://api.themoviedb.org/3/movie/top_rated: Películas mejor valoradas.
* https://api.themoviedb.org/3/movie/upcoming: Películas a estrenar.
* https://api.themoviedb.org/3/search/movie: Obtiene las películas buscadas por el usuario.
* https://api.themoviedb.org/3/movie/{movie_id}: Obtiene los detalles de una película.
* https://api.themoviedb.org/3/movie/{movie_id}/watch/providers: Obtiene los proveedores de streaming de una película.
* https://api.themoviedb.org/3/movie/{movie_id}/images: Obtiene las imágenes de una película.
# YouTube:
* Link: https://developers.google.com/youtube/v3?hl=es-419
### Restricciones de uso:
* API usa una cuota para garantizar que los desarrolladores usen el servicio según lo previsto y no creen aplicaciones que reduzcan de manera injusta la calidad del servicio o limiten el acceso de otras personas.
* Los proyectos que habilitan la API de datos de YouTube tienen una asignación de cuota predeterminada de 10,000 unidades por día.
### Implementación:
* https://www.youtube.com/embed/${trailer.key}: Obtiene el tráiler de una película.
