# DagonFilms: Web de peliculas y series.
El objetivo principal de DragonFilms es proporcionar a los usuarios una web centralizada donde puedan descubrir, revisar y seguir películas y/o series de su interés. Busca ofrecer una experiencia intuitiva y personalizada que fomente la interacción de los usuarios y facilite la exploración del vasto mundo del cine.

### Características Principales:
* Exploración de Películas: Los usuarios pueden explorar una amplia selección de películas y series disponibles en la plataforma, con información detallada proporcionada por la API de TheMovieDB. Pueden buscar por título.

* Creación de Reseñas: Los usuarios pueden crear reseñas personalizadas para cualquier película o serie en la plataforma. Pueden compartir sus opiniones, calificaciones y comentarios con la comunidad de DragonFilms.

* Visualización de Tráileres: Los usuarios tienen acceso a tráileres y avances de películas o series directamente desde la plataforma, gracias a la integración de la API de YouTube. Esto les permite tener una vista previa de las películas antes de verlas.

* Gestión de Favoritos: Los usuarios pueden agregar películas y/o series a su lista de favoritos para acceder fácilmente a ellas más tarde. Esta función permite a los usuarios mantener un registro de lo que desean ver o que les han gustado.

* Visualización de Reseñas Anteriores: Los usuarios pueden ver sus reseñas previas en su perfil. Esto les permite recordar lo que han revisado y ver cómo han evolucionado sus opiniones a lo largo del tiempo.

### Tecnologías Utilizadas:
* Para la realización correcta de la plataforma web se utilizan los reconocidos lenguajes HTML y CSS (Front End), PHP (Back End), además de JS. 

* API de TheMovieDB: Se integra la API de TheMovieDB para obtener información detallada sobre películas, incluyendo sinopsis, reparto, puntuaciones de usuarios y más.

* API de YouTube: Se utiliza la API de YouTube para acceder a tráileres y avances de películas directamente desde la plataforma DragonFilms.
